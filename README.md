# Novel Analysis DB — BL/TL作品分析ツール

BL・TL（女性向け）ポルノ小説の創作にあたり、売れている作品の構造を分析し、その結果を保存・管理するWebアプリケーション。

## 設計思想

**「AIとAIを繋ぐ中間保存庫」**

ユーザーがこのアプリから分析対象の本文をエクスポートしてClaudeに渡し、Claudeが返した分析結果をこのアプリに保存し、後日また別のClaude対話に参考資料としてエクスポートする。

## 機能

### 画面1：分析対象作品リスト
- 登録済み作品をカード形式で表示
- ジャンルタグ（BL/TL）・作品タグ・話数・分析ステータス表示
- 分析ステータス3段階（未着手／分析中／分析完了）

### 画面2：作品メタ情報＆本文入力
- 作品基本情報の登録・編集
- 話数連動アコーディオン式本文入力（折りたたみ）
- 話数を減らしても入力内容を保持

### 画面3：分析結果保存＆エクスポート
- **機能①** 分析依頼エクスポート（構成分析／読者の声／キャラクター抽出）
- **機能②** 分析記録の保存（タグ付き・日付自動付与・時系列表示）
- **機能③** 再エクスポート4種（文章分析用／プロット用／キャラ決め用／全データ）

## 技術仕様

- **フロントエンド**: 単一HTMLファイル（依存なし）
- **データ保存**: IndexedDB（ブラウザローカル）
- **認証**: 不要
- **デザイン**: ダークモード + ディープパープルグロウ

## 使い方

`index.html` をブラウザで開くだけで動作します。
